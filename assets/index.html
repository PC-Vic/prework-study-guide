Prework Study Guide

HTML

  • the character set UTF 8 tells the browser how to understand and display the characters
  • Head Element contains information about the webpage.
  • Body Element represents the visible content shown to the user
  • Footer Element define information that typically appears at the bottom of a document, such as contact info

CSS

  • Cascading Style Sheets
  • Used to format and style webpages
  • Selectors target specific HTML elements
  • Properties determine the visual presentation or behavior of the selected elements
  • Values are assigned to CSS properties to define specific styles

JavaScript

  • Variables are used to store and manipulate data
  • Data Types Numbers, Strings, Booleans, Arrays, Objects, Null, Undefined, Symbol
  • Functions are blocks of reusable code that perform a specific task
  • Control Flow the order in which statements are executed

Git

  • Git helps you keep track of changes in your code and allows you to save different versions of your code over time
  • CD command change current directory
  • ls command lists the contents of current directory
  • git status This command shows the current state of your Git repository
  • pwd command displays the path of the current directory you are in
  • git init initializes a new Git repository in your current directory. It sets up the necessary files and folders for Git to start tracking changes in your code
  • git add -A commit all files in the working branch
  • git add stages changes in your code to be included in the next commit
  • git commit creates a new commit, saving the staged changes to your Git repository
  • git push upload your local commits to a remote repository, such as GitHub or GitLab
  • git checkout -b switch to newly created branch
  • git main switch to main branch
  • git pull fetch and merge the latest changes from the remote main branch
  • git checkout -a inspects all branches
